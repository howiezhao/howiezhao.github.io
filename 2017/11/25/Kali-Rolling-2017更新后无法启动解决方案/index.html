Kali Rolling 2017更新后无法启动解决方案

Posted on 2017-11-25 | In Bug |

BusyBox v1.27.2 (Debian 1:1.27.2-1) built-in shell (ash)

有时更新Kali后重新启动会出现如下显示，并无法进入系统界面

解决方法：

  1. 在此界面输入blkid命令，查看自己的分区号
  2. 依据自己的分区号输入fsck -y /dev/sda1，我的分区号是sda1。
  3. 输入exit命令退出重启电脑
